# Templates - Proyectos Pre-configurados

Templates disponibles para el skill app-developer:

## 1. SaaS Starter

**Para**: Producto SaaS
**Incluye**: Auth, DB, Stripe, Dashboard

## 2. E-commerce

**Para**: Tienda online
**Incluye**: Carrito, Products, Checkout

## 3. Blog/CMS

**Para**: Blog o contenido
**Incluye**: Posts, Categories, Admin

## 4. API REST

**Para**: Solo backend
**Incluye**: Auth, CRUD, Docs

## 5. Real-time

**Para**: Chat, notificaciones
**Incluye**: WebSockets, Redis

## Usage

```bash
/app-developer new mi-app --template saas
/app-developer new mi-tienda --template ecommerce
```
